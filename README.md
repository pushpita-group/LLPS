This repository presents a minimal continuum framework for studying re-entrant liquid–liquid phase separation (LLPS) in biomolecular condensates regulated by small molecules. The model combines Cahn–Hilliard phase-field dynamics with concentration-dependent Flory interactions to capture biphasic condensate behavior, where low modulator concentrations promote phase separation while higher concentrations suppress it. The framework reproduces experimentally observed condensate formation, dissolution, morphology transitions, and coarsening kinetics in systems such as Tau–tannic acid and TDP-43–bis-ANS. The repository includes numerical simulations, analytical tools, and visualization scripts for exploring phase behavior and condensate dynamics in soft matter and biophysical systems.

The file named "Code" contains all of the necessary Python scripts required for the Project.
